zroger PEAR channel

Registering the channel: 
pear channel-discover zroger.github.com/pear
Listing available packages: 
pear remote-list -c zroger
Installing a package: 
pear install zroger/package_name
Installing a specific version/stability: 
pear install zroger/package_name-1.0.0
pear install zroger/package_name-beta
Receiving updates via a feed: 
http://zroger.github.com/pear/feed.xml
